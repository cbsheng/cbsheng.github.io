从硬件出发，浅谈操作系统的段机制与页机制
  • June 16, 2015
  • OS  
  • memory management  

我们写一个程序，经过编译之后会变成一堆的指令。操作系统在执行这个程序的时候，也正是执行这堆指令。 指令可以是 取数据的指令 或 取下一条被执行指令 的指令。但无论是什么指令，都会进行内存的访问和寻址。就像是我们必须找到东西放在哪里，我们才能去使用它。

那操作系统怎样帮我们去寻址呢？答案是 内存寻址涉及到内存的 段机制 和 页机制。
